# Водокачка+

## О проекте

```
В билллинговую систему "Водокачка+" заносится тариф, и ежемесячно — показания. Также есть редактируемый список подключенных к водокачке дачников с необходимой для расчетов информацией. На основе имеющихся данных производится расчет и каждому дачнику выставляется счет. Проект выполнен vue3 и laravel10.
```

## Установка

```
1. git clone https://github.com/arkadykorsakov/wizard.git
2. cd wizard
3. composer install
4. php artisan migrate --seed
5. npm install
```

### Запуск

```
npm run serve
php artisan serve
```
